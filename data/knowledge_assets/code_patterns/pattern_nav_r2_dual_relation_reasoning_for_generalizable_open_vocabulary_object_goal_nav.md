# Open-vocabulary object-goal navigation agents fail to generalize to unseen objects and environments due to lack of explicit relational reasoning between target objects and scene context.

**Domain**: `Planning_Decision`

## Fix

Use a dual-relation reasoning framework with a Chain-of-Thought (CoT) pipeline that first models target-environment relations (e.g., object co-occurrence, spatial priors) and then environment-action relations (e.g., feasible exploration steps), fine-tuned on synthetic CoT data generated by Qwen2.5-VL-7B.

## Anti-pattern

Standard end-to-end VLN policies that directly map observations to actions without explicit relational reasoning.

## Cross-domain analogies

- **Perception_Vision** → Use a generative model with relational reasoning loss to directly map observations to goal-directed actions.
  - related fix: Use a GAN with a compressed sensing loss to directly reconstruct images from undersampled k-space data in a single feedforward pass.
- **Learning_Training** → Generate synthetic scene-object relational data at scale to train navigation agents.
  - related fix: Train a transformer agent on 4.2 million synthetic instruction-trajectory pairs generated at scale, reducing reliance on human demonstrations.
- **Control_Locomotion** → Use a benchmark with standardized relational reasoning tasks to train navigation agents on object-scene context.
  - related fix: Use EB-Manipulation benchmark to evaluate and train agents on low-level actions (joint torques, end-effector poses) with standardized tasks that require precise perception and spatial reasoning.
